# Student Performance Prediction

## Description
This project predicts student performance based on academic and personal factors using machine learning. 
It identifies students at risk and helps in suggesting interventions.

## Dataset
The dataset used is `student_performance_dataset.csv` containing features like:
- Gender
- Study time
- Previous grades
- Family support
- etc.

Sample data is available in the folder. For full dataset

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
